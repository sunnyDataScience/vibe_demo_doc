# 資料卡生成 Prompt
## 客觀事實描述，無臆測分析

---

## 🎯 一鍵 Prompt

```
請為這個資料集生成一張「資料卡」，只描述客觀事實，不做推測或分析建議。

檔案：[你的檔案路徑]

**輸出格式（固定）：**

## 📊 資料卡

**一句話描述：** [客觀描述這是什麼資料]

### 基本資訊
- **檔案大小**: [筆數] 筆記錄
- **欄位數量**: [數量] 個欄位
- **時間範圍**: [起始日期] 至 [結束日期]
- **主要內容**: [記錄什麼事物]

### 欄位結構
| 欄位名稱 | 資料型態 | 非空值數量 | 唯一值數量 | 備註 |
|---------|---------|-----------|-----------|------|
| [列出所有欄位的基本統計] |

### 資料特徵
- **完整性**: [缺失值情況]
- **重複值**: [是否有重複記錄]
- **數值範圍**: [數值欄位的最小/最大值]
- **分類分布**: [類別欄位的主要類別]

### 資料品質
✅ **良好**: [沒有問題的部分]
⚠️ **注意**: [需要留意的問題]
🚫 **缺失**: [明顯缺失或異常的部分]

---
**生成時間**: [當前時間]
**檔案路徑**: [原始路徑]
```

**要求：**
- 只報告客觀統計事實
- 不提供分析建議
- 不推測資料用途
- 不評估商業價值
```

---

## 📋 使用方法

### 1. 複製 Prompt
複製上面的整個 prompt 內容

### 2. 替換檔案路徑
把 `[你的檔案路徑]` 換成實際檔案位置

### 3. 丟給 AI
貼給任何 AI 工具，等待資料卡輸出

### 4. 獲得結果
得到純客觀的資料描述卡片

---

## 🎯 輸出特色

### ✅ 只包含事實
- 檔案基本統計
- 欄位結構描述
- 數值分布範圍
- 缺失值情況

### ❌ 不包含推測
- 不猜測資料用途
- 不建議分析方向
- 不評估商業價值
- 不提供解讀建議

---

## 📊 範例輸出

### 輸入：
```
檔案：Taiwan_SuperMarket_Sales_2025.csv
```

### 輸出：
```
## 📊 資料卡

**一句話描述：** 包含交易ID、日期、金額等欄位的表格型資料

### 基本資訊
- **檔案大小**: 1,000 筆記錄
- **欄位數量**: 29 個欄位
- **時間範圍**: 2025-01-01 至 2025-03-30
- **主要內容**: 每筆記錄包含交易識別、時間、金額、分類等資訊

### 欄位結構
| 欄位名稱 | 資料型態 | 非空值數量 | 唯一值數量 | 備註 |
|---------|---------|-----------|-----------|------|
| Invoice ID | String | 1,000 | 1,000 | 無重複值 |
| Date | Date | 1,000 | 89 | 時間分布連續 |
| Sales | Float | 1,000 | 990 | 數值範圍320-35252 |
| Branch | String | 1,000 | 7 | 7個不同類別 |

### 資料特徵
- **完整性**: 所有欄位無缺失值
- **重複值**: Invoice ID 無重複，其他欄位可能重複
- **數值範圍**: Sales (320-35252), Quantity (1-10)
- **分類分布**: Branch 有7個類別，分布相對均勻

### 資料品質
✅ **良好**: 無缺失值，時間連續，ID唯一
⚠️ **注意**: 部分數值欄位範圍較大
🚫 **缺失**: 無明顯缺失問題

---
**生成時間**: 2024-12-24 15:30
**檔案路徑**: Taiwan_SuperMarket_Sales_2025.csv
```

---

**原則：客觀描述事實，讓資料自己說話。**
