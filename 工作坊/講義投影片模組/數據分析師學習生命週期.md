## Level 1：數據素養階段（0-6個月）🌱

### 核心能力
```
【必備技能】
✅ Excel 基礎操作
✅ 基本統計概念
✅ 簡單圖表製作
✅ SQL 查詢基礎
```

### 學習重點
| 領域 | 具體內容 | 目標水準 |
|:-----|:---------|:---------|
| 數據理解 | 資料類型、資料結構 | 能辨識和處理 |
| 統計基礎 | 平均數、中位數、標準差 | 理解並應用 |
| 視覺化 | 長條圖、圓餅圖、折線圖 | 選對圖表類型 |
| 商業思維 | 理解KPI、提出好問題 | 能轉譯業務需求 |

### 產出能力
```
可以完成：
- 基礎業務報表
- 簡單資料清理
- Excel儀表板
- 數據品質報告
```

## Level 2：分析師階段（6-18個月）📊

### 核心能力
```
【進階技能】
✅ Python/R 程式設計
✅ 統計推論和假設檢定
✅ 進階SQL（窗口函數、CTE）
✅ 數據視覺化工具（Tableau/PowerBI）
```

### 學習重點
| 領域 | 具體內容 | 目標水準 |
|:-----|:---------|:---------|
| 程式設計 | Python Pandas、NumPy | 能處理複雜數據 |
| 統計分析 | 假設檢定、相關分析、迴歸 | 能設計實驗 |
| 商業智慧 | Tableau、Power BI | 建立互動式儀表板 |
| 分析方法 | A/B測試、同期群分析、RFM | 獨立完成專案 |

### 產出能力
```
可以完成：
- 業務洞察報告
- A/B測試設計與分析
- 預測模型原型
- 自動化報表系統
- 客戶分群分析
```

### 學習路徑
```python
analyst_path = {
    "第1季": "Python基礎 + Pandas",
    "第2季": "統計推論 + A/B測試",
    "第3季": "SQL進階 + 資料倉儲",
    "第4季": "Tableau/PowerBI + 儀表板設計",
    "第5季": "時間序列分析 + 預測",
    "第6季": "完整專案實戰"
}
```

### 專案建議
```
1. 客戶流失預測（Churn Analysis）
   - 數據：電信/訂閱服務數據
   - 方法：邏輯迴歸、決策樹
   - 產出：預警系統

2. 銷售預測模型
   - 數據：歷史銷售數據
   - 方法：時間序列、ARIMA
   - 產出：月度預測報告

3. 用戶行為分析
   - 數據：網站/App日誌
   - 方法：漏斗分析、路徑分析
   - 產出：優化建議
```

### 里程碑檢核
```
□ 能用Python處理百萬級數據
□ 完成至少3個端到端分析專案
□ 設計並執行A/B測試
□ 建立可分享的GitHub作品集
□ 通過一項專業認證
```

## 學習心法 🧘

### 高效學習原則
```
1. 專案驅動學習
   不要只看教程，要動手做專案

2. 輸出倒逼輸入
   教學相長，寫文章加深理解

3. 社群互助成長
   加入社群，互相學習

4. 持續迭代改進
   定期回顧，調整學習策略

5. 平衡廣度深度
   T型人才：一專多能
```

### 避免的陷阱
```
❌ 教程收集症：收藏很多，看得很少
❌ 完美主義：等準備好才開始
❌ 單打獨鬥：不交流不分享
❌ 忽視基礎：急於學習高階技術
❌ 紙上談兵：只學理論不實踐
```
